How to install Extension

  1. Login to Magento backend.
  2. Go to System > Tools > Compilations.
    If Compiler Status is Enabled, disable the compilation by clicking the button Disable in the upper right corner.
  3. Backup your store database and web directory.
  4. Download extension from your Account.
  5. Unzip the extension locally.
  6. Copy contents of the folder to the root directory of Magento store.
  7. Completely clear the site cache.
    Go to System > Cache Management. Click button Flush Magento Cache.
    If you want to clear the cache manually, you need to remove a folder /var/cache at the server.
  8. Logout from Magento backend and login again.
  9. If compilation was enabled before installation, you need to run recompilation process to enable compilation.
    Go to System > Tools > Compilation and click on Run Compilation Process.
  10. Go to System > Configuration > YouTube Gallery and open Google Developers Console.
  11. Create a new project in Google Developers Console
  12. In Project Name set "YouTube Gallery", next copy your project ID, then click in Create button.
  13. In Magento paste your project ID in "Project ID" field in API Settings tab.
  14. In Google Developers Console click in APIs & Auth > APis, next click "YouTube Data API".
  15. Click Enable API button.
  16. Click in APIs & Auth > Credentials, next click Add credentials button, then click API key.
  17. Click Server key button.
  18. In name put YouTube Gallery Server, next click Create button.
  19. Copy API key that was generated, next click "Ok" button.
  20. In Magento paste API Key in "API Key for server applications" field.
  21. In Google Developers Console click Add credentials button, next click API key.
  22. Click Brownser key button.
  23. In Project Name set YouTube Browser, next click Create button.
  24. Copy API key that was generated, next click "Ok" button.
  25. Copy API key that was generated, next click "Ok" button.
  26. In Magento paste API Key in "API Key for browser applications" field.
  27. In Google Developers Console click Permissions button, next copy first email in Account.
  28. In Magento paste Email in "Service account Email Address" field.
